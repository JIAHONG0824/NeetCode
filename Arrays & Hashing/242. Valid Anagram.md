## 排序
```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        # 最佳
        if len(s) != len(t):
            return False
        # 最差
        return sorted(s) == sorted(t)
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
## 用空間換取時間
```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        if len(s) != len(t):
            return False
        count_s = defaultdict(int)
        count_t = defaultdict(int)
        for i in range(len(s)):
            count_s[s[i]] += 1
            count_t[t[i]] += 1
        return count_s == count_t
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
