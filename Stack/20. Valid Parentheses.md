## 堆疊
```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        match = {")": "(", "}": "{", "]": "["}
        for c in s:
            if c == "(" or c == "{" or c == "[":
                stack.append(c)
            elif len(stack) > 0 and stack[-1] == match[c]:
                stack.pop()
            else:
                return False
        return False if len(stack) > 0 else True
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(n)$
