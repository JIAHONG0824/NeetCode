## 反轉+雙指針
```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        k=k%len(nums)
        l,r=0,len(nums)-1
        while l < r:
            nums[l],nums[r]=nums[r],nums[l]
            l+=1
            r-=1
        l,r=0,k-1
        while l < r:
            nums[l],nums[r]=nums[r],nums[l]
            l+=1
            r-=1
        l,r=k,len(nums)-1
        while l < r:
            nums[l],nums[r]=nums[r],nums[l]
            l+=1
            r-=1
        return
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
