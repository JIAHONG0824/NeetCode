## 利用額外空間
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        left = [0] * len(height)
        right = [0] * len(height)
        ans = 0
        max_height = 0
        for i in range(1, len(height)):
            max_height = max(max_height, height[i - 1])
            left[i] = max_height
        max_height = 0
        for i in range(len(height) - 2, -1, -1):
            max_height = max(max_height, height[i + 1])
            right[i] = max_height
        for i in range(len(height)):
            min_height = min(left[i], right[i])
            if min_height > height[i]:
                ans += min_height - height[i]
        return ans
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(n)$
