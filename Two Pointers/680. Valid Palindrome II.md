## 雙指針
```python
class Solution:
    def validPalindrome(self, s: str) -> bool:
        l, r = 0, len(s) - 1
        while l < r:
            if s[l] != s[r]:
                left, right = l + 1, r
                while left < right:
                    if s[left] != s[right]:
                        break
                    else:
                        left += 1
                        right -= 1
                if left >= right:
                    return True
                left, right = l, r - 1
                while left < right:
                    if s[left] != s[right]:
                        return False
                    else:
                        left += 1
                        right -= 1
                return True
            else:
                l += 1
                r -= 1
        return True
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
