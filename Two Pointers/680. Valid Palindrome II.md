## 雙指針
```python
class Solution:
    def validPalindrome(self, s: str) -> bool:
        l, r = 0, len(s) - 1
        while l < r:
            if s[l] != s[r]:
                # 試著刪除左邊
                left, right = l + 1, r
                while left < right and s[left] == s[right]:
                    left += 1
                    right -= 1
                if left >= right:
                    return True
                # 試著刪除右邊
                left, right = l, r - 1
                while left < right and s[left] == s[right]:
                    left += 1
                    right -= 1
                return left >= right
            l += 1
            r -= 1
        return True
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
