## 雙指針
```py
class Solution:
    def validPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1
        while left < right:
            if s[left] != s[right]:
                l, r = left + 1, right
                while l < r and s[l] == s[r]:
                    l += 1
                    r -= 1
                if l >= r:
                    return True
                l, r = left, right - 1
                while l < r and s[l] == s[r]:
                    l += 1
                    r -= 1
                return True if l >= r else False
            else:
                left += 1
                right -= 1
        return True
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
