## 雙指針
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        write=0
        for read in range(len(nums)):
            if nums[read]!=nums[write]:
                write+=1
                nums[read],nums[write]=nums[write],nums[read]
        return write+1
```
* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
