## 雙指針
```python
class Solution:
    def mergeAlternately(self, word1: str, word2: str) -> str:
        l = r = 0
        string = []
        while l < len(word1) and r < len(word2):
            string.append(word1[l])
            l += 1
            string.append(word2[r])
            r += 1
        while l < len(word1):
            string.append(word1[l])
            l += 1
        while r < len(word2):
            string.append(word2[r])
            r += 1
        return "".join(string)
```
* 時間複雜度: $O(m+n)$
* 空間複雜度: $O(m+n)$
